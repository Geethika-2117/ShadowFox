# LoanWise AI

AI-powered Loan Approval Prediction System built using Machine Learning.

## Features

- Loan Approval Prediction
- Explainable AI Decisions
- Modern Dashboard UI
- Logistic Regression Model
- Real-time Prediction

## Technologies

- Python
- Flask
- Scikit-Learn
- Pandas
- HTML
- CSS

## Dataset

Loan Prediction Dataset

## Model Performance

Accuracy: 86.18%
F1 Score: 90.81%

## Run Locally

pip install -r requirements.txt

python app.py

Open:
http://127.0.0.1:5000

The model evaluates:

- Credit History
- Income
- Loan Amount
- Dependents
- Employment Status
- Property Area

to determine loan approval eligibility.

Approval Factors:

✓ Good Credit History
✓ Stable Income
✓ Affordable Loan Amount
✓ Positive Financial Profile
